# Ambiguity-set supervision

An ordinary observation has one approved full-decomposition target. Both K=2
hypotheses are supervised toward it and receive a concentration penalty. Each
member of a validated observational near-collision pair has the same two-target
set: its own decomposition and the paired approved decomposition. Alternate
source ordering uses a frozen minimum-cost two-permutation coordinate mapping.

Only pairs passing partition isolation, four-group disjointness, exact replay,
additivity, finite-array and hash checks, observation similarity, target
divergence, and rescaling-artifact rejection enter a target set. Group IDs are
provenance only and never inference inputs. Weak pairs cannot pad the count.

All 2,000 Thayer-MH pairs passed construction, but the trained decoder still
produced zero own, alternate, and both-mode scientific coverage. This concerns
the shared token-decoder mechanism, not the validity of the approved target
sets, and it does not establish source uniqueness.

Thayer-LG confirmed that every exact approved set is representable, prompt-
swap consistent, forward-plausible, and covered. It also found that the frozen
objective ranks a compromise below the exact set on every ambiguous microset
row. The approved targets remain valid; their optimized objective geometry is
misaligned with their scientific coverage geometry.

Thayer-SA left these target sets and the hard two-permutation rule unchanged.
Its surrogate ranked approved truth above compromises, but the official
free-output optimizer reached only 25% both-mode coverage from the collapsed
mean and 31.25% from the wrong source allocation. The target contract remains
valid; objective optimization is still insufficient.
