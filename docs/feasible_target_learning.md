# Feasible-target learning

Feasible-target learning uses offline outputs already inside the unchanged
scientific acceptance region as training-only representatives. It does not
rename projections as astronomical truth, change the approved target sets, or
provide truth or constraint values to inference.

Thayer-FP froze P0 homotopy targets only after exact-truth feasibility,
determinism, canonical hashing, ordinary/ambiguous set coverage, and
forward-evaluation gates passed. Model inputs remained the normalized g/r/z
blend and Condition-C coordinate prompt. The loss contained only direct
requested-source and companion-source reconstruction under the fixed hard
two-permutation match. Scientific surrogate, forward, source-sum, prompt-swap,
concentration, diversity, uncertainty, and Atlas losses were absent.

The unchanged 165,612-parameter Thayer-ME lowered direct target loss to
0.001951 but did not enter scientific coverage. All ordinary, own, alternate,
and both-mode rates stayed at zero. The result is therefore a neural
feasibility failure, not evidence that the projected targets were invalid.
Strong prompt swap and forward consistency do not substitute for truth
coverage.
