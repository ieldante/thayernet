# Empirical ambiguity witness

For one observed blend and prompt, retain candidate decompositions that pass
the frozen truth-free forward-consistency tolerance. A witness exists when at
least two retained requested-source layers differ beyond a frozen scientific
image, flux, color, centroid, or valid shape limit and the difference is not an
output-contract artifact.

A witness falsifies practical uniqueness within the tested finite candidate
set and fixed forward model. It does not enumerate every explanation. Failure
to find a witness does not prove recoverability or uniqueness.

Atlas v0 separates two results. Constructed truth decompositions form witnesses
on 50/50 frozen noisy observations, directly demonstrating ambiguity for those
cases. The available model candidates form witnesses on only 19/50. On a
deterministic 25-scene validation control comparison, model-candidate diameter
fails the frozen operational baseline gate: AUROC 0.4712 and zero Atlas recall
at 4% control false positives. Therefore the empirical witness is scientifically
valid as an exhibited counterexample, but candidate diameter is not a usable
Thayer-Audit gate in v0.

Family/checkpoint identity, target truth, source identity, true error, true SNR,
true obstruction, and simulator difficulty remain forbidden auditor inputs.
