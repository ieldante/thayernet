# L0 Decoder Optimization Diagnosis

Square D0 and D1 are directly navigable for the frozen one-scene target set,
while square D2 final-head-only fitting fails all scientific coverage gates.
This establishes a frozen-feature conditioning problem but does not determine
whether the existing full L0 decoder can reshape its penultimate features into
a feasible region.

The first square-only D3 campaign produced no scientific result because its
required D1 feature endpoint was not persisted. Thayer-D1R has now replayed D1
exactly and persisted that endpoint with fresh-process and batch-invariance
proofs. It did not instantiate or optimize a decoder body.

The next justified experiment is one separately preregistered one-scene
square-only D3 campaign using the same cached prompt features, initial L0 expert
states, frozen targets, direct objective, hard assignment, mapping, and
evaluation gates. No capacity increase, broader data, or protected partition
is authorized by D1R alone.
