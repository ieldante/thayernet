# Z-band capacity diagnostics

Thayer-FP's projection result reproduced: z-band flux was the limiting
scientific constraint on 173 of 256 homotopy entries, and the median P0 alpha
and normalized correction remained 0.999979 and 0.946369.

Thayer-CL produced no z-band-by-capacity result because the output-mapping
uniqueness gate failed before model construction. There is therefore no basis
for attributing z-band error to decoder width, no capacity curve, and no
smallest passing condition. Z-band metrics must remain mandatory evaluation
outputs in the separate output-parameterization campaign and in any later
authorized width ladder.
