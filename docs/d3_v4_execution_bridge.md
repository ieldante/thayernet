# D3 v4 execution bridge

Execution bridge v4 is append-only and reference-only. It copies no scientific
array value or threshold.

Its frozen authority precedence is:

1. Bundle v3 for policy, continuation, semantic states, outcomes, and
   authorization.
2. Bundle v2, resolved only through v3, for architecture, artifact schemas,
   model/optimizer construction, and numerical execution.
3. Capsule v1 for scientific values, tolerances, units, bands, and evaluator
   configuration.
4. Runtime readiness for process and filesystem isolation.

synthetic-campaign scope and is non-authoritative for v4. The bridge freezes
`automatic_scientific_continuation = true_after_v4_preflight`.
