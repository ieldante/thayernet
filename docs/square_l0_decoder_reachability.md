# Square L0 Decoder Reachability

Square D0 and D1 remain authoritative 100/100/100 passes, while square D2
remains an authoritative 0/0/0 failure. Thayer-D1R provides a complete
successful feature endpoint. Thayer-D3R did not test whether the existing two
46,470-parameter L0 decoders can reach a successful feature region from the
immutable cached prompt features.

The retry stopped before any decoder gradient or update because an import
dependency attempted a guard-prohibited deletion and the temporary-directory
bootstrap could not complete under the no-delete contract. This is an
execution-readiness failure, not evidence for decoder optimization, capacity,
hard assignment, or square-map conditioning.

Neither a square-only eight-scene campaign nor a decoder-capacity ladder is
authorized.

that the isolated allowed evidence did not persist the exact scientific sky
vector and plausibility thresholds needed to evaluate novel D3 outputs. It
